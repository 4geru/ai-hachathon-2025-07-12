# 技術的な挑戦と革新
### 「Sky Canvas」を支える最新テクノロジー

```mermaid
graph TD
    A[スマートフォン<br/>センサー取得] -->|リアルタイム送信| B[Supabase Realtime<br/>WebSocket通信]
    B -->|瞬時通知| C[ディスプレイアプリ<br/>花火描画]
    C -->|高品質レンダリング| D[大画面ディスプレイ<br/>p5.js + WebGL]
    A -->|触覚フィードバック| E[バイブレーション<br/>Vibration API]
    C -->|没入感| F[音響効果<br/>Web Audio API]
```

#### 🛠️ 技術スタック
*   **Frontend**: React + Next.js 14, TypeScript, p5.js
*   **センサー**: DeviceMotion/Orientation API (加速度・角度)
*   **Backend**: Supabase (PostgreSQL + Realtime)
*   **描画**: p5.js + WebGL による高性能レンダリング
